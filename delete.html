حذف بيانات المستخدم

نرحب بطلبات حذف البيانات. لو حابب نحذف أي بيانات مرتبطة بحسابك أو باستخدامك لهذا التطبيق، من فضلك اختر واحدة من الطرق التالية وابعتلنا المعلومات المطلوبة.

مهم: بعد استلام طلب حذف البيانات، هنراجع الطلب ونقوم بحذف البيانات في مدة لا تتجاوز 7 أيام عمل. بعض العمليات (زي إزالة النسخ الاحتياطية) قد تستغرق وقتًا أطول وفق سياسات الأنظمة المعنية.

طرق تقديم طلب الحذف

في الرسالة اذكر: اسم الحساب/البريد المستخدم، وصف مختصر للطلب، وأي معلومات تساعدنا نحدد حسابك (مثلاً معرف الصفحة أو رابط الملف الشخصي).

لو عندك واجهة أمامية أو API بتتعامل مع الطلبات، نستخدم endpoint لحذف البيانات. مثال على طلب JSON (مُعياري):

POST /api/delete-data
Content-Type: application/json

{
  "user_id": "123456789",
  "email": "user@example.com",
  "reason": "أريد حذف بياناتي"
}

لو عايز نفعل endpoint زي ده، ابعتلي وانا أساعدك تجهّز الكود البسيط.

ماذا يتم حذفه؟

هنقوم بحذف أية بيانات شخصية مخزنة مرتبطة بالمعرف اللي قدمته مثل الاسم، الإيميل، وأي بيانات مرتبطة بالتفاعل مع التطبيق. نحتفظ فقط بسجلات تقنية ضئيلة إن كانت لازمة للامتثال القانوني أو الأمان (ونوضح ده لو حصل).

تأكيد الإجراء

بعد تنفيذ الحذف، هنرسل لك رسالة تأكيد على الإيميل اللي قدمته تفيد إن البيانات اتحذفت بالفعل.

لو محتاج مساعدة فنية في تنفيذ endpoint أو تريد أن أجهز لك صفحة HTML أو كود Flask/Node.js يعالج طلب الحذف ويجيب JSON، قولّي وانا أحضرهولك جاهز للرفع.
